# remnawave-profile-editor-ui
Modern visual editor for Remnawave profiles and Xray configurations. Supports VLESS, VMess, Trojan, REALITY, HTTP/SOCKS Proxy, Routing and JSON validation.
# Xray Config Editor

🇬🇧 English | 🇷🇺 Русский

A modern React-based visual editor for Xray and Remnawave configurations.

Designed to simplify creation, editing, validation, and visualization of complex Xray configurations without manually editing JSON files.

---
## 🚀 Live Demo

**Try it now — no installation required!**

👉 **[https://remnawave-profile-editor-ui.vercel.app](https://remnawave-profile-editor-ui.vercel.app)**

Open your browser, edit Xray/Remnawave configs visually, and export JSON instantly. Works everywhere.

## Installation

```bash
git clone https://github.com/yourname/xray-config-editor.git

cd xray-config-editor

npm install

npm run dev
```

or

```bash
yarn install
yarn dev
```

* Built-in validation

## Зачем нужен этот проект?

Ручное редактирование конфигураций Xray занимает много времени и часто приводит к ошибкам.

Этот редактор позволяет:

* Работать через удобный интерфейс
* Проверять конфигурации автоматически
* Визуализировать маршруты
* Быстро создавать профили
* Упрощать сопровождение больших конфигов
